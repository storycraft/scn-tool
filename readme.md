# scn-tool
M2 scn scenario file text & string extract / insert tools.

## Usage
### Extract
```
scn-script-extractor.exe <scn_file> [output_file]
```

### Patch
```
scn-script-patch.exe <json_patch_file> <scn_file> [output_file]
```
